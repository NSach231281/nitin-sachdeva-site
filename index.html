Decision Philosophy

How I Think About Problems

  • Decisions matter more than predictions — a perfect model feeding a bad policy is worse than a simpler model with a good decision framework.
  • Constraints define reality; they cannot be an afterthought in model design.
  • Uncertainty must be explicit — not buried in point estimates that leadership mistakes for certainty.
  • Models must survive governance and adversarial conditions, not just validation metrics on historical data.
  • The most valuable thing I can leave behind is not a model — it is a team that can maintain, question, and improve the system.

DESC-PHIRO: Demand Forecasting & Inventory Optimisation System

End-to-end hybrid ARIMA + GBT + LSTM demand forecasting system deployed across a 3,500+ node multi-echelon supply network. Integrated hierarchical reconciliation (MinT), automated reforecast cycles, drift monitoring, and champion/challenger model management. Fed directly into multi-objective inventory optimisation (Pyomo + Gurobi + Benders decomposition) minimising cost, fill rate violation and lead time SLA simultaneously.

Supply Chain · TVS Motors

SCM_Inventory: Multi-Echelon Inventory Optimisation Engine

Production multi-objective stochastic inventory optimisation framework for a 3,500+ node multi-echelon network. Formulated as a two-stage stochastic program with Benders decomposition for scalability. Simultaneously minimises holding cost, transportation cost and SLA violation probability while accounting for demand uncertainty and lead time variability via scenario-based simulation (SAA).

₹300 Cr cumulative savings · Custom (s,S) policies per SKU-node · Service level improvement across all tiers

Agentic AI · TVS Motors

Agentic Planning Intelligence Framework

Production multi-agent orchestration framework for supply chain planning. Orchestrator agent decomposes planning objectives into subtasks routed to specialist agents (demand signal, inventory position, supplier capacity, constraint validation). RAG-powered planning assistant using GPT-4 + Pinecone + LangChain enables natural-language querying of supply chain data. Three-layer guardrail architecture prevents hallucinated replenishment decisions.

35% improvement in discoverability · Exception management automation · Planner decision cycle acceleration
